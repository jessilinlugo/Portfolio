My Collection of Works

Will It Stream is a collaborative project I recently helped complete. Once a user has inputted a tv or movie title, two separate APIs (OMDb and uTelly) are called to display the relevant information. A poster, title, critics score, release date, runtime, and a brief description are all displayed as a result of the search. As well as any applicable streaming service icons leading directly to the title on the corresponding streaming platform. The User Interface is designed to mimic the cinema experience. With so many theaters shutting down, and our bedrooms and living rooms doubling as home theaters, Will It Stream was created for the home viewer in mind.

Week Day Scheduler was an exercise in JavaScript in which I really shined. The application is essentially a daily planner, in which to lay out and schedule your day. The planner runs on a timer and will change the color of the text box based on the time of day. The hours that have passed will be highlighted a light yellow, the current hour is a muted red, and the remaining hours of the day are a soothing green. I didn't want the color palette to be overwhelming for users, and felt the muted winter's day aesthetic was perfect for the short days and long nights.

jessilinQuery was an exercise in JavaScript in which I really struggled. The application is a game which the user clicks a start button, and is presented with a question and 4 clickable answer buttons. Selecting one of the answer buttons will take you to the next page which displays a different question. There are 5 questions to click through, and once you finish, you're supposed to be presented with a Game Over page, displaying your score. However, it took me about 8 or 9 hours just to get to the point of having the screen display the 1st question. Then an additional hour with a tutor to be able to run through the following questions. My ultimate goal is to be able to add a timer, score counter to keep track of the correct answers as you run through the quiz, and make your choices. Although truthfully, because it's been so difficult, the little successes have been that much more rewarding.
